# v1 + v2 — Fases 0 al 3 (todas implementadas)

All phases 0 through 3 are shipped and in production as of May 2026.

## Fase 0 — Landing + Cotizador

- **Landing** (`landing/`) — Astro 6 + Tailwind. Sections: Hero slideshow, Servicios, Planes, Memoriales, Convenios, Contacto, Ubicación. Navbar with "Ingresar" dropdown → portals.
- **Cotizador** (`cotizador/`) — React + Vite embeddable iframe. 8-step quote flow. Prices fetched live from `GET /api/configuracion/servicios`. Currently commented out in `landing/src/pages/index.astro` — activate when crematory opens.
- **`/acceso`** — Portal selector page (Mi portal / Portal socios) linked from landing navbar.

## Fase 1a — CRM Core

Clients, pets, services, prepaid plans, leads, agenda, roles.

Key modules:
- **Servicios** — 5-state lifecycle: `pendiente → en_proceso → listo → entregado / cancelado`. 3 types: cremación individual, cremación comunitaria, entierro. 3 responsible staff fields.
- **Planes** — Graduated coverage: 0% (cuotas 1–6), 50% (7–12), 100% (13+). Mascota adicional = +50% cuota.
- **Leads** — 6-state kanban. Cron: nuevo +72h → perdido, interesado +48h → perdido, perdido +10 days → deleted. Round-robin assignment.
- **Agenda** — Calendar filtered by rol.
