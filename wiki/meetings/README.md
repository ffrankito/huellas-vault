# Meetings

Dated meeting notes. Decisions, action items, open questions surfaced per session.

## Naming convention

`YYYY-MM-DD-[topic].md` — e.g. `2026-05-06-kickoff-linear.md`

## What belongs here

- Sync notes with Huellas de Paz
- Internal Ravenna planning sessions
- Any meeting that produced a decision or action item

## Lifecycle

- Open questions → open a note in `gaps/`
- Decisions made → create an ADR in `decisions/`
- Action items → create a Linear issue and link it here
