# Runbooks

Operational step-by-step guides. For when something needs to be done carefully and correctly.

## What belongs here

- Deploy process (Vercel, Supabase)
- How to run a manual DB migration
- How to add a new user to the CRM
- How to recover from a failed deployment
- How to rotate credentials

## Format

Each runbook should have: **When to use**, **Prerequisites**, **Steps** (numbered), **Verification**.
